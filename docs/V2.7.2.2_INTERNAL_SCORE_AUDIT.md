# V2.7.2.2 — Internal Score Audit

## الهدف
تفسير الدرجة الداخلية لكل فرصة دون تغيير الأوزان أو حد التأهل ودون اختراع بيانات مالية.

## مخرجات التدقيق
لكل فرصة يسجل التقرير:

- الدرجة النهائية.
- الحد المطلوب للبحث الخارجي.
- الفجوة بين الدرجة والحد.
- نقاط كل مكوّن متاح داخل `score_components`.
- مجموع المكونات قبل أي Gate أو Cap.
- أسباب الدرجة من `score_reasons`.
- قرار المحرك الاقتصادي السابق.
- عدد الأدلة الناقصة.
- تشخيص واضح لأسباب انخفاض الدرجة.

## التشخيصات الأساسية

- `missing_total_score`
- `missing_score_components`
- `no_verified_economics_points`
- `no_evidence_points`
- `missing_evidence_blocks_financial_validation`
- `upstream_evidence_gate_active`
- `component_total_differs_from_final_score_due_to_gate_or_cap`
- `below_external_research_threshold`

## أمر التشغيل

```bash
PYTHONPATH=src python scripts/run_internal_score_audit.py \
  data/todays_opportunities.json \
  --output data/validation/v2.7.2.2-internal-score-audit.json
```

## خارج النطاق

- لا يخفض حد 60 تلقائيًا.
- لا يغيّر أوزان التقييم.
- لا يمنح نقاطًا عند غياب الأدلة.
- لا يرسل فرصة ضعيفة إلى Brave بالقوة.
- لا يصدر قرار شراء أو تواصل آلي.

بعد قراءة التقرير نقرر هل المشكلة في البيانات الداخلة، أو في مكوّن بعينه، أو في Gate اقتصادي يمنع الدرجة النهائية من الارتفاع.
