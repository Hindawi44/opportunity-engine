# V2.6.4 — External Evidence Loop

## الهدف

ربط البحث الخارجي بملف الفرصة الحي دون تحويل النظام إلى مشترٍ آلي أو أداة تواصل آلي.

## الدورة

1. قراءة `Living Investment File`.
2. تحديد فجوات البحث الحقيقية فقط.
3. إنشاء استعلام سوق أو مشترين عند الحاجة.
4. استخدام موصل البحث الخارجي مع الـCache وحدود الطلبات المطبقة داخله.
5. تمرير النتائج إلى Market Comparables Engine أو Buyer Discovery Engine.
6. تحويل النتائج المقبولة فقط إلى Evidence.
7. تقييم الدليل وحفظه في Evidence Store.
8. إعادة تشغيل Scenario Generator فقط عند إنشاء دليل جديد أو تحديث ملاحظة قائمة.

## قواعد السلامة

- فشل Brave أو Rate Limit يسجل كخطأ ولا يوقف معالجة الفرصة.
- لا يعتبر ظهور شركة في البحث إثباتًا على نية شراء.
- لا يرسل النظام بريدًا أو رسالة أو اتصالًا.
- لا يصدر قرار شراء نهائيًا.
- لا يعاد البحث نفسه لنفس الفرصة داخل دورة التشغيل.
- لا يعاد توليد السيناريوهات إذا لم تتغير الأدلة.
- القيم المفقودة لا تتحول إلى صفر.

## نطاق V2.6.4

يشمل Orchestrator قابلًا للحقن والاختبار. الربط المجدول مع Daily Pipeline وصفحة العرض يأتي في المرحلة التالية بعد ثبات هذه الدورة.
