# Opportunity Engine

محرك أولي لتسجيل فرص الأعمال والمزادات في النرويج، حساب التكلفة الكلية، تقدير الربح، وتصنيف الفرص إلى:

- 🟢 فرصة قوية
- 🟡 تحتاج مراقبة
- 🔴 لا تستحق

## المتطلبات

- Python 3.11 أو أحدث
- pip أو uv لإدارة الحزم

## التثبيت

```bash
# تثبيت المكتبات المطلوبة
pip install -r requirements.txt
```

## التشغيل

### تطبيق Streamlit (الواجهة الرسومية)

```bash
streamlit run dashboard.py
```

سيتم فتح التطبيق تلقائياً في متصفحك على `http://localhost:8501`

### مثال تشغيل مباشر

```bash
python run.py
```

## الاختبارات

```bash
pytest
```

لتشغيل الاختبارات مع تفاصيل:

```bash
pytest -v
```

## هيكل المشروع

- `dashboard.py`: تطبيق Streamlit - الملف الرئيسي للتطبيق.
- `run.py`: مثال تشغيل مباشر.
- `run_pipeline.py`: تشغيل خط أنابيب معالجة البيانات.
- `src/opportunity_engine/`: منطق التقييم والحساب.
  - `models.py`: نماذج البيانات (Opportunity, Evaluation).
  - `evaluator.py`: محرك التقييم والحساب.
  - `io.py`: قراءة وكتابة ملفات CSV.
  - `auksjonen_import.py`: استيراد ملفات من نمط Auksjonen النرويجي.
  - `report.py`: إنشاء التقارير.
- `config/settings.json`: إعدادات افتراضية قابلة للتعديل.
- `tests/`: اختبارات آلية شاملة.
- `.gitignore`: ملفات مستثناة من التحكم بالإصدار.

## الاستخدام

### عبر التطبيق (Streamlit)

1. شغّل `streamlit run dashboard.py`
2. أدخل بيانات الفرصة في النموذج
3. اضغط "تحليل الفرصة" للحصول على التقييم
4. سيتم عرض التكلفة الكلية والربح المتوقع والعائد والحد الأقصى للمزايدة

### البيانات المدخلة

- **اسم الفرصة**: وصف الفرصة
- **رابط المزاد**: رابط الإعلان (اختياري)
- **سعر المزايدة/الشراء**: المبلغ المدفوع للمزاد
- **عمولة المزاد**: رسوم الوسيط
- **الاستكاليف الإضافية**: النقل، الفك، التخزين، الإصلاح، وغيرها
- **قيمة إعادة البيع المحافظة**: السعر المتوقع للبيع
- **مستوى المخاطرة**: من 1 (منخفضة) إلى 5 (عالية جداً)
- **هامش الربح المستهدف**: النسبة المئوية للربح المرغوب

## الصيغة الحسابية

### التكلفة الكلية

```
التكلفة الكلية = سعر المزايدة + العمولة + النقل + الإصلاح + الفك + التخزين + التكاليف الأخرى + ضريبة القيمة المضافة
```

### الربح المتوقع

```
الربح = قيمة إعادة البيع - التكلفة الكلية
```

### العائد على الاستثمار

```
العائد (%) = (الربح / التكلفة الكلية) × 100
```

### الحد الأقصى للمزايدة

يحسب بناءً على هامش الربح المستهدف، مع احتساب ضريبة القيمة المضافة إذا كانت تنطبق على سعر المزايدة.

## معايير التصنيف

- **🟢 فرصة قوية**: عائد ≥ 35% مع مستوى مخاطرة ≤ 2
- **🟡 تحتاج مراقبة**: بين القوية والضعيفة
- **🔴 لا تستحق**: ربح ≤ 0 أو مستوى مخاطرة ≥ 5
